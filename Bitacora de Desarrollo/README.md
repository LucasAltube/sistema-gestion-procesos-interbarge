# Bitácora de Desarrollo

Esta carpeta concentra todo lo relativo a:
- Cambios realizados por el asistente (bitácora de sesión).
- Notas técnicas para commits / Git / GitHub.
- Archivos auxiliares generados para análisis.

## Archivos
- `REGISTRO_DE_SESION.md`: registro cronológico de acciones.
- `ANALISIS_GIT_GITHUB.md`: criterios sugeridos para versionado/commits.
- `indicadores_v3_extraido.csv`: extracción de indicadores desde v3 para análisis.
